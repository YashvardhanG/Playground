Information

General:
• To add a note click on the 'Add a note' button
• Delete a note by double clicking on it
• Edit a note by single clicking on it
• All notes are auto-saved

Keyboard Commands:
• Add a bullet list by typing /b and hitting enter
• Add an ordered list by typing /o and hitting enter
• Create a note by pressing { key
• Delete last note by pressing } key
• Toggle Info Corner with \ key
